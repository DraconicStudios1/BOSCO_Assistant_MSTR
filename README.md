# BOSCO BETA V0.0.1

BOSCO is an Assistant application capable of reading .json files to display schedules and calculate the time until certain timestamps.
The main focus of the BOSCO beta branch will be integration with the Crosstalk Solutions Project NOMAD server system, which can contain Map data, The entirety of Wikipedia,
and a Large Language Model, which uses only natively stored data to train and function.

I will be embedding a Text To Speech engine into BOSCO to use the outputs from the Project NOMAD LLM to speak. BOSCO may undergo ***A NAME CHANGE*** over the course of the Beta Branch.
